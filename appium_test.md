## Appium
* 开源跨平台、多语言支持的移动应用自动化工具
* 测试App运行平台
    * ios
    * Android
    * Windows
    * FirefoxOS
### 自动化程序语言
    python、java、Ruby、Js、php、c#
### 测试程序运行平台
    Windows，Linux、Mac OS
### 需要的库
* Appium-Python-Client
* selenium
### 安装Appium Server
* 方法一：node.js & npm
    * npm install -g appium
* 方法二：Appium Desktop
